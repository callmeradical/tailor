Why Tailor

Everything you need.
Nothing you don't.

Tailor handles the setup work that costs you a day every time you switch machines.

⚡

One-command bootstrap

A single curl command installs Ansible, clones your config, and applies it. Go from unboxed to productive in minutes.

📦

Ansible-native

No new DSL. Your config repo is a valid Ansible project. Use any Ansible module, role, or collection — the full ecosystem is available.

🔄

Idempotent by design

Run tailor apply as often as you like. Already-correct state is untouched. Only drift gets fixed.

🪄

Adopt ad-hoc changes

tailor adopt brew ripgrep captures what you installed manually back into your declared config — one package at a time.

🔍

Safe dry-run

tailor apply --check shows exactly what would change without touching your machine. Always safe to run.

🗂️

Version-controlled config

Your entire machine configuration lives in a git repo. Full history, diffs, rollbacks, and sharing — all included.

How it works

Ansible under the hood.
Tailor on top.

Tailor wraps ansible-playbook with a friendlier CLI and a curated starter template. Your config repo is the source of truth.

packages

Installs Homebrew formulae and casks declared in group_vars/all.yml. Supports taps.

dotfiles

Clones your existing dotfiles repo and runs its installer. Works with any dotfiles setup — no migration needed.

macos-prefs

Applies macOS system preferences via osx_defaults: dock, keyboard repeat, trackpad, screenshots, and more.

vscode

Installs VS Code extensions by ID. Skips gracefully if the code CLI is not installed.

hooks

Runs pre-apply.sh and post-apply.sh shell scripts around the apply lifecycle. No-op if absent.

your roles

It's just Ansible. Add your own roles, pull from Ansible Galaxy, or write custom tasks — no restrictions.

CLI Reference

Simple commands.
Powerful results.

All commands are thin wrappers around Ansible. Nothing magic.

tailor apply
Converge your machine to its declared state. Runs ansible-playbook site.yml against localhost.
tailor apply --check
Dry-run. Shows exactly what would change without touching anything. Always safe.
tailor apply --role <name>
Apply a single role by tag name. e.g. --role packages or --role macos-prefs.
tailor validate
Syntax-check the playbook. Parse-only — no task evaluation, no system changes.
tailor adopt brew <package>
Add an already-installed Homebrew package to group_vars/all.yml. Idempotent. Opens file in $EDITOR.
tailor adopt file <path>
Copy a config file into the dotfiles role and register a symlink task. Idempotent. Opens task file in $EDITOR.
tailor --usage
Print an agent-oriented briefing — structured usage instructions for AI agents and automation tools.
